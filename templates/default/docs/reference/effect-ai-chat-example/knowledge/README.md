# Knowledge

Start with `patterns/effect-skill-index.md` in the generated project. This
directory is copied reference material for module-level Effect skills, UI rules,
and custom lint examples from the applied Effect AI example repo.

## Rules

| File                                                       | Purpose                                                    |
| ---------------------------------------------------------- | ---------------------------------------------------------- |
| `knowledge/rules/dashboard-ui.md`                          | Dashboard and app shell layout guidance.                   |
| `knowledge/rules/effect-atom.md`                           | Repo specific rules for Effect Atom and Effect Form state. |
| `knowledge/rules/form-validation-message.md`               | Form validation and validation message guidance.           |
| `knowledge/rules/accessible-notifications-and-messages.md` | Notification and message accessibility guidance.           |

## Skills

Synced module guides live under `knowledge/skills/`.

Read `knowledge/skills/README.md` to see the available Effect module guides, then open the relevant file for the module you are changing.
