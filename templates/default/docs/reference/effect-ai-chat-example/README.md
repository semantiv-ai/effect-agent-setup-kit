# Effect AI Chat Example Knowledge

This directory contains curated reference material copied from the local
`effect-ai-chat-example` reference repo.

Use it as agent-readable local context:

- `knowledge/skills/` contains module-level Effect v4 guides.
- `knowledge/rules/` contains UI and Effect Atom rules from the applied app.
- `scripts/oxlint-rules/` contains custom oxlint rule examples.
- `oxlintrc.example.json` shows the applied app's broader lint profile.

These files are reference material, not generated application source. Do not
import code from this directory. If a copied rule or script is useful, adapt it
into the project deliberately and add a verifier probe when it becomes part of
the setup contract.

Source:

- [`lucas-barake/effect-ai-chat-example`](https://github.com/lucas-barake/effect-ai-chat-example)
